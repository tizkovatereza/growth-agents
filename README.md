# Custom Droids

A collection of custom [Factory droids](https://docs.factory.ai) for reusable AI workflows.

## chart-maker-formal-svg

Generates clean, formal SVG charts for blog posts and articles. Produces timeline charts, pie charts, bar charts, and conceptual diagrams with a consistent minimal style.

**Color palette:** Warm beige background, dark navy accents, white text on headers.

**Style:** No overlapping text, readable labels with generous padding, visible but subtle axes, less is more.

### Usage

Copy the droid file to your Factory droids directory:

```bash
cp chart-maker-formal-svg.md ~/.factory/droids/
```

Then invoke it:

```bash
droid chart-maker-formal-svg
```
